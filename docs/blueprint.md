# **App Name**: Buildoholics HMS

## Core Features:

- Reservation Management: Allow staff to create, view, update, and cancel hotel reservations.
- Customer Management: Manage customer profiles, including contact information and reservation history.
- Room Management: Track room availability and status in real-time.
- Automated Suggestion: Suggest room upgrades and additional services to customers based on the customer's profile and the booking details as a tool.
- Reporting: Generate reports on bookings and availability.

## Style Guidelines:

- Primary color: Strong red (#D11141) to convey energy and boldness.
- Background color: Off-white (#F2F2F2) to maintain neatness and cleanliness in a light theme.
- Accent color: Light red (#F08080) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque sans-serif, will give a modern and neutral style for information and headlines.
- Use minimalist icons with red accents.
- Grid-based layout for a structured and neat appearance.
- Subtle transitions and animations for a smooth user experience.
